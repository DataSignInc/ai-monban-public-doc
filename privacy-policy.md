# AI MONBAN プライバシーポリシー
========

制定日：2026年6月6日

更新履歴：本プライバシーポリシーの更新履歴はGitHubで公開しておりますので、こちらの[GitHubリポジトリの変更履歴](https://github.com/DataSignInc/ai-monban-public-doc/commits/main/privacy-policy.md)からご確認ください。

### はじめに

本プライバシーポリシーは株式会社DataSign（以下「当社」と言います）の提供するAI MONBAN（以下「本サービス」と言います）における個人情報の取り扱いについて記載しています。

本サービスは、企業データと生成AI・AIエージェントとの接続を安全かつ統制された形で仲介する「Trusted Gateway」として、お客様（契約組織。以下「テナント」と言います）の業務でご利用いただくことを想定したサービスです。本サービスにおいて取得する個人情報のうち、テナント内のユーザーに関する情報については、原則としてテナントが個人情報取扱事業者として責任を負い、当社はテナントから委託を受けて取り扱います。

当社全体の個人情報保護方針については、[こちらの個人情報保護方針](https://datasign.jp/pii_policy/)をご確認ください。

#### 3）本サービスがAIに連携するデータの取り扱いについて

本サービスは、テナント管理者が許可した範囲においてのみ、ユーザーから指定された外部リソース上のデータをAIに連携します。<br>
当該データの内容は、当社が独自に二次利用することはなく、AIモデルの学習等にも使用しません。連携データの一部は「データアクセスログ」として記録されますが、当該ログはテナント管理者の監査用途のために保管され、テナント単位で隔離して管理されます。

### 3.個人情報の開示・訂正等について

1. 本サービスで取得した個人情報については、それぞれ以下の方法で開示・訂正が可能です。

| 個人情報の種類 | 開示・訂正方法 |
---- | ----
| メールアドレス | 本サービスログイン後のアカウント情報画面より確認いただけます。修正をご希望の場合はテナント管理者または当社までお問合せください。 |
| パスワード | 本サービスログイン後のアカウント情報画面、またはパスワードリカバリフローより修正いただけます。パスワードはソルト付きハッシュ化されているため、確認はいただけません。またハッシュ化後の値については、セキュリティ上の理由により開示できません。 |
| ユーザー名・所属組織・割り当てロール／グループ／権限 | 本サービスログイン後の管理画面（テナント管理者）より、確認・修正いただけます。一般ユーザーがご自身の情報の確認・修正をご希望の場合は、所属するテナント管理者にお問合せください。 |
| 外部リソース（MCPサーバー）への接続情報 | 本サービスログイン後のConnections画面より、接続の有無の確認および削除（接続解除）が可能です。トークン自体はセキュリティ上の理由により開示できません。 |
| データアクセスログ | 本サービスログイン後のData Access Log画面より、テナント管理者は当該テナントに属する全ユーザーのログを確認できます。一般ユーザーはご自身のログを確認できます（権限設定による）。 |
| 操作ログ | 本サービスログイン後のOperation Log画面より、テナント管理者は当該テナントに属する全ユーザーのログを確認できます。一般ユーザーはご自身のログを確認できます（権限設定による）。|
| 決済情報 | クレジットカード支払いの場合、本サービスログイン後のアカウント情報画面より、一部確認・訂正いただけます。クレジットカード番号全体は当社では保持していないため、ご確認いただけません。その他、氏名、会社名、住所、電話番号等の確認・修正についてはお問合せください。 |

2. 本サービスで取得した個人情報の削除の請求がある場合、その請求がご本人様（またはテナント管理者）であることが確認でき次第、速やかに情報を削除いたします。なお、データアクセスログは、テナントとの契約に基づきテナント管理者の監査用途のために保存されているため、削除はテナント契約者からのご請求に基づき行います。

3. 本サービスのデータアクセスログは、ホットデータ（直近3ヶ月分、PostgreSQL）とコールドデータ（全期間、S3）の2層構成で保管しており、それぞれテナントとの契約に定める保管期間に従って管理しています。

### 5.ご利用のブラウザから自動的に送信される情報について

1. 当社では、本サービスおよび[AI MONBANウェブサイト](https://ai-monban.com/)において、ブラウザから送信される以下の情報を取得しています。
    * IPアドレス
    * 閲覧したサイトのURL、リファラ（検索ワードを含みます）、アクセス日時
    * ブラウザのユーザーエージェント（OSの情報も含まれます）
    * ブラウザのcookie情報（ご利用のブラウザを識別するための識別子を含みます）

2. 上記の取得したデータは以下の目的で利用されます。
    * 個人情報と組み合わせた操作ログの分析
    * 本サービスの利用状況の把握
    * 本サービスサイト外での広告の配信、広告効果の測定

3. これらの目的のため当社は第三者のサービスを利用しております。第三者のサービスの一覧と、パーソナルデータの取り扱いについては、[オンラインプライバシー通知](https://as.datasign.co/privacy/policy/ai-monban-ja)をご確認ください。

4. 第三者サービスへのこれらの情報の送信は、下部に表示されている「あなたのプライバシー」をクリックして立ち上がるツールによって制御できます。
